`pacman`常用命令

| 命令            | 解释                       |
| --------------- | -------------------------- |
| pacman -Syyu    | 更新软件源                 |
| pacman -S xxx   | 安装软件                   |
| pacman -Rs xxx  | 在删除包的同时，也删除依赖 |
| pacman -Ss xxx  | 搜索含关键字的包           |
| pacman -Scc xxx | 清理所有的文件             |
