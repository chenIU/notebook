## MySQL 日期函数

| 格式 | 描述                  |
| ---- | --------------------- |
| %Y   | 年，4位               |
| %y   | 年，2位               |
| %M   | 月名                  |
| %m   | 月，数值（00-12）     |
| %d   | 月的天，数值（00-31） |
| %e   | 月的天，数值（0-31）  |
| %H   | 小时（00-23）         |
| %h   | 小时（01-12）         |
| %I   | 小时（01-12）         |
| %i   | 分（00-59）           |
| %S   | 秒（00-59）           |
| %s   | 秒（00-59）           |
