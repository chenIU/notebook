| 命令  | 压缩                     | 解压缩               |
| ----- | ------------------------ | -------------------- |
| zip   | zip xxx.zip xxx          | unzip xxx.zip        |
| gzip  | gzip xxx                 | gunzip xxx.gz        |
| rar   | rar a xxx.rar xxx        | rar x xxx.rar        |
| tar   | tar -zcvf xxx.tar.gz xxx | tar -zxvf xxx.tar.gz |
| bzip2 | bzip2 -z xxx             | bzip2 -d xxx.bz2     |



**注意事项**

gzip不支持压缩文件夹
