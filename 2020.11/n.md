# node版本管理工具n使用指南

## 安装

```bash
npm install -g n
```



## 常用命令

| 命令              | 解释                   |
| ----------------- | ---------------------- |
| n                 | 查看所有已安装的版本   |
| n xxx             | 安装某个版本           |
| n latest          | 安装最新的版本         |
| n statble         | 安装稳定的版本         |
| n rm xxx          | 移除某个版本           |
| n use xxx some.js | 以执行的版本来执行脚本 |
